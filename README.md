# Icertis - Contract Intelligence Clone

✨ A professional and responsive enterprise landing page clone of Icertis. Showcasing advanced CSS layout techniques, AI-driven intelligence sections, and a premium corporate experience design.

## 🔗 Live Demo
Check out the live project here: [108580-icertis.netlify.app](https://108580-icertis.netlify.app/)

---

## 🚀 Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Smart Navigation**: Modern sticky navigation bar with a responsive hamburger menu for mobile devices.
- **Vera AI Foundation**: Dedicated sections showcasing AI-driven contract intelligence and agentic workflows.
- **Enterprise Showcase**: Professional layouts for case studies (Microsoft, etc.), analyst reports (Gartner, Forrester), and industry solutions.
- **Interactive UI**: Subtle micro-animations, hover effects on cards, and smooth scrolling for a premium feel.
- **Multi-page Experience**: Comprehensive site structure including:
  - **Landing Page**: The core value proposition.
  - **Blog**: Insights and latest news layout.
  - **Research Center**: Educational resources and reports.
  - **Contact**: Clean, professional lead generation form.

---

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3**: Vanilla CSS for layout (Flexbox/Grid), custom animations, and responsive design.
- **Assets**: SVG icons and optimized WebP/PNG images for fast loading.
